61 /*
62  * The selectivity of Boolean operators is essential in order to determine
63  * efficient execution plans for queries. The temporal extension defines
64  * several classes of Boolean operators (equal, less than, overlaps, ...),
65  * currently 30, each of which can have as left or right arguments a built-in
66  * type (such as integer, timestamptz, box, geometry, ...) or a newly defined
67  * type (such as period, tint, ...), currently 33.
68  *
69  * There are currently 3,392 operators, each of which is identified by an Oid.
70  * To avoid enumerating all of these operators in the Oid cache, we use a
71  * two-dimensional array containing all possible combinations of
72  * operator/left argument/right argument (currently 23 * 40 * 40 = 36,800 cells).
73  * The invalid combinations will be initialized to 0.
74  */
